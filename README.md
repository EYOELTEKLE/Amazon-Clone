# Concepts used

- Server Side Rendering

- Conditional Rendering

- Customizable Utility Classes

- Async and Promise for Fetching API endpoints

## Using npm

Run commands

1) ```npm install```


2) ```npm run dev```


## Or using yarn

Run commands 

1) ```npm install --global yarn```

2) ```yarn install```

3) ```yarn run dev```
